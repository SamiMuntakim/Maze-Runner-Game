# Maze-Runner-Game
The Labyrinth of Legends

A Java based maze exploration game featuring a custom built GUI and interactive pathfinding mechanics. The project demonstrates object oriented design, event driven programming, and rendering logic using standard Java libraries.

Features

Interactive 2D maze rendered through a custom GUI.

Player movement with real time collision detection.

Dynamic maze loading from text based layouts.

Clear separation of game logic, UI components, and data structures.

Simple yet expandable architecture for future levels or AI pathfinding.

Tech Stack

Java

Swing/AWT for GUI components

To compile and run the Graphical User Interface (GUI),
ensure that the libraries 'JavaFX SDK', the 'JavaFX' user library, and 'JRE System Library' 
are in the module buildpath and run the "Game.java" file and interact with the window that will appear.
